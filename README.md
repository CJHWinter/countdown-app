# 倒计时氛围感 ⏰

一个精美的双模式计时网页应用，提供专注倒计时和实时时钟功能，内置500+条励志名言，具有出色的视觉效果和音效反馈。

## ✨ 核心功能

### 🎯 专注模式
- **精确倒计时**: 支持天/时/分/秒显示，误差控制在毫秒级别
- **分组快捷设置**: 常用时长、快速专注、深度专注三大分组，涵盖10分钟到8小时
- **增强时间选择**: 美化的日期时间选择器，支持实时预览和剩余时间计算
- **进度可视化**: 动态进度条和百分比显示
- **氛围音效**: 心跳音效营造紧张感，完成时播放成功音效
- **完成庆祝**: 光爆动画、震动反馈（支持设备）
- **智能继续**: 完成后可快速继续15分钟专注，自动停止音乐进入专注状态
- **奖励音乐系统**: 完成后播放选定的奖励音乐，支持搜索和筛选

### 🕐 日常时钟模式
- **北京时间显示**: 自动从多个时间API获取标准时间
- **智能校准**: 每5-10分钟自动校准，误差≤1秒
- **离线降级**: API失败时自动切换到本地时间
- **数字翻牌效果**: 每个数字独立的翻转动画
- **秒数控制**: 可选择显示/隐藏秒数
- **实时同步状态**: 显示最后同步时间和状态

### 📜 名言系统
- **海量名言**: 内置500+条中文励志名言
- **智能分类**: 按"时间"、"专注"、"努力"、"成功"等标签过滤
- **多种切换方式**: 手动切换、自动轮播、随机显示
- **打字机效果**: 优雅的文字显示动画
- **收藏功能**: 本地持久化收藏，支持导出
- **复制分享**: 一键复制名言到剪贴板

### 🎨 主题系统
- **18种精美主题**: 6种普通主题 + 12种节日主题
- **自定义背景图片**: 每个主题支持自定义背景图片
- **4种数字风格**: 科技感、可爱圆润、优雅细线、俏皮泡泡
- **实时预览**: 主题选择页面实时预览效果
- **一键切换**: 点击主题按钮快速切换
- **智能保存**: 主题偏好自动保存到本地

### 🎵 音乐系统
- **双模式音乐**: 专注完成奖励音乐和休息背景音乐分离管理
- **智能缓存**: 音乐列表自动保存到本地存储，避免重复扫描
- **智能搜索**: 实时搜索音乐文件，快速定位所需曲目
- **拓展音乐库**: 加宽音乐列表显示，支持更多音乐展示
- **左侧控制面板**: 独立的音乐控制图标，可随时收起/展开播放器
- **模式智能切换**: 不同模式间自动适配音乐控制显示
- **文件夹指引**: 明确的音乐文件夹路径提示，便于用户管理音乐
- **缓存管理**: 提供清除音乐缓存功能，支持重新扫描音乐文件

### 🔊 音效系统
- **多种音效**: 滴答声、心跳声、完成提示音
- **音量控制**: 独立音量调节和静音开关
- **场景化音效**: 不同情况播放对应音效
- **可配置性**: 每种音效都可单独开关

### 📊 使用统计系统
- **自动记录**: 每次专注/休息会话自动记录，无需手动操作
- **数据持久化**: 使用浏览器localStorage自动保存，关闭浏览器后数据依然保留
- **多维度统计**: 
  - 总专注时长、总休息时长
  - 完成率统计
  - 连续使用天数
  - 最近20条使用记录
- **可视化图表**: 使用ECharts展示
  - 📈 每日专注时长趋势（最近7天）
  - 📊 专注 vs 休息时间占比（饼图）
  - 📅 每周专注时长分布（柱状图）
  - 🕐 时段分布（24小时使用频率）
- **数据导入/导出**: 
  - 支持导入JSON格式的统计数据（覆盖或合并模式）
  - 支持导出JSON格式的统计数据备份
  - 数据格式验证，确保导入数据正确性
- **数据管理**: 可随时清除所有统计数据

### 📅 我的定制倒计时系统
- **多日期管理**: 支持创建和管理多个自定义倒计时日期
- **灵活设置**: 每个日期可设置：
  - 日期名称（如"我的生日"、"考研倒计时"等）
  - 开始时间（可选，默认为当前时间）
  - 结束时间（必需）
  - 对自己想说的话（可选，个性化激励语）
- **精美展示**: 
  - 长条卡片格式显示激励话语，与里程碑标签分离
  - 进度环可视化显示已过时间百分比
  - 四个里程碑标签：100天、50天、30天、最后一周
  - 日期切换卡片，方便在多个日期间切换
- **日期卡片**: 主页面以卡片网格形式展示所有日期，一目了然
- **数据导入/导出**: 
  - 支持导出所有自定义日期到JSON文件
  - 支持导入JSON格式的日期数据（覆盖或合并模式）
  - 智能去重，避免重复导入相同日期
  - 数据格式验证，确保导入数据正确性
- **数据持久化**: 所有日期数据自动保存到本地存储

#### ⚠️ 批处理文件编码问题说明

如果您是从GitHub直接下载的ZIP压缩包（非git clone），可能会遇到以下情况：

- ✅ `创建桌面快捷方式.bat` 正常显示（纯ASCII字符）
- ❌ `安装依赖.bat` 和 `生成音乐列表.bat` 出现中文乱码或秒退

**问题原因：**
- 原始bat文件使用UTF-8编码（包含`chcp 65001`命令）
- Windows默认使用ANSI/GBK编码打开bat文件
- 从GitHub下载的ZIP压缩包会保留原始编码，导致中文显示异常

**解决方案（按推荐度排序）：**

**方案一：使用英文版（最可靠）**
- `安装依赖_英文版.bat` - 完全英文界面，避免编码问题
- `安装依赖_极简版.bat` - 极简版本，使用最基本的ASCII字符
- `生成音乐列表_最终版.bat` - 英文版音乐列表生成工具

**方案二：手动执行（最稳定）**
如果所有bat文件都有问题，可以直接在命令提示符中手动执行：

```bash
# 1. 安装Python依赖
pip install pillow
cd crawl_videos
pip install -r requirements.txt
cd ..

# 2. 生成音乐列表
node generate-music-list.js

# 3. 启动应用
python -m http.server 8000
```

**方案三：转换编码（临时方案）**
使用支持UTF-8的文本编辑器（如VS Code）打开原文件，然后另存为ANSI编码

### ⚠️ 常见问题和注意事项

#### Q1: 为什么双击 index.html 看不到音乐？

**A:** 应用必须通过 HTTP 服务器访问。

**解决：** 使用 `quickstart.bat` 或任何启动脚本。

#### Q2: 启动后命令行窗口能关闭吗？

**A:** 不能！关闭窗口会停止服务器。

**解决：** 
- 最小化窗口
- 或使用后台运行方式（高级用户）

#### Q3: 端口 8000 被占用怎么办？

**A:** 修改启动脚本中的端口号：
```bash
# 将 8000 改为其他端口，如 8080
python -m http.server 8080
```

#### Q4: 音乐文件太大能上传到 GitHub 吗？

**A:** 不建议！

- GitHub 单个文件限制 100MB
- 仓库建议 < 1GB
- 音乐文件会被 `.gitignore` 自动忽略

**建议：**
- 本地使用：保留音乐文件在本地
- 在线使用：使用云存储（阿里云OSS）提供音乐 URL

#### Q5: 如何更新音乐列表？

**A:** 
1. 添加音乐文件到 `music/学习/` 或 `music/休息/`
2. 双击 `生成音乐列表.bat`
3. 在应用中点击 🔄 刷新按钮

#### Q6: 视频无法拖动进度条？

**A:** 视频拖动需要服务器支持 HTTP Range 请求。

**解决方案：**
- ✅ **推荐**：安装 Node.js，`quickstart.bat` 会自动使用
- ✅ 手动启动：`node server.js`
- ⚠️ Python 的 `http.server` 不支持视频拖动

**如何安装 Node.js：**
1. 访问：https://nodejs.org/
2. 下载并安装 LTS 版本
3. 重新运行 `quickstart.bat`

**技术说明：**
- `server.js` 已支持 HTTP Range 请求（206 Partial Content）
- `quickstart.bat` 优先使用 Node.js 服务器
- Python 服务器不支持 Range，视频无法拖动

## ⚙️ 配置说明

### 时间校准策略
应用使用多级时间校准策略确保精确度：

1. **主要API**: `worldtimeapi.org/api/timezone/Asia/Shanghai`
2. **备用API**: `timeapi.io` 和 `time.is`
3. **降级方案**: 本地 `Intl.DateTimeFormat` with `timeZone: 'Asia/Shanghai'`
4. **校准频率**: 每5分钟自动校准一次
5. **错误处理**: API失败时自动切换到下一个或本地时间

### 数据持久化

## 🐛 故障排除

### 常见问题

#### 1. 时间显示不准确
- **检查**: 网络连接是否正常
- **解决**: 点击"同步时间"手动同步
- **原因**: 时间API可能暂时不可用

#### 2. 音效无法播放
- **检查**: 音频文件是否存在于正确路径
- **解决**: 参考音频文件说明获取音效
- **原因**: 浏览器可能阻止了自动播放

#### 3. 名言无法加载
- **检查**: `quotes.json` 文件是否存在
- **解决**: 确保文件路径正确且格式有效
- **原因**: 文件路径或JSON格式问题

#### 4. 动画卡顿
- **检查**: 浏览器性能和其他标签页占用
- **解决**: 在设置中启用"减少动画"
- **原因**: 设备性能或浏览器资源不足

#### 5. 移动端显示异常
- **检查**: 浏览器是否为现代浏览器
- **解决**: 更新浏览器到最新版本
- **原因**: CSS Grid/Flexbox兼容性问题

### 调试技巧
1. 打开浏览器开发者工具查看错误信息
2. 检查网络标签页确认API请求状态
3. 查看控制台日志了解应用状态
4. 清除localStorage重置设置: `localStorage.clear()`
